About HimBuddy

HimBuddy is an advanced environmental monitoring system using ESP32. It detects fire, landslides, and vibrations (earthquakes) using sensors and communicates alerts via Bluetooth and Firebase to the Android application. It also logs events to SD card and can trigger emergency notifications. The Android app and ESP32 code can be downloaded using the buttons above.
